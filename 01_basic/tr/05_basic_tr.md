## basic_tr22.py

次のプログラムがあります。

``` python

```

次の実行結果となるようにPythonプログラムを作成してください。

### 実行結果

* `X: ` に `3` 、 `Y: ` に `7` を指定した場合

```

y: 7
*******
*******
*******
```

> 変数 `x` は行数（縦に出力する数）、変数 `y` は列数（横に出力する数）を意味します。
