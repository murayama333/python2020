## `no35.py`

## `no40.py`

次のプログラムがあります。

``` py
# TODO def contains function

print(names)
print("Bob:", contains(names, "Bob"))
print("Dave:", contains(names, "Dave"))
```

実行結果のとおり出力してください。

＜実行結果＞

``` 
['Andy', 'Bob', 'Carol']
Bob: True
Dave: False
```
