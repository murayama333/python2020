# Hello Python!

### hello.py

``` python
print("Hello World")
```

### 実行

``` 
$ python hello.py
Hello World
```

---

### 解説

* テキストエディタで作成したPythonプログラムは拡張子を.pyとして保存する
* ターミナル（コマンドプロンプト）でpythonコマンドにファイル名を指定して実行する
* print関数によって画面に文字列データを出力できる
