# エクササイズ - データ構造 - タプル

## tuple_ex1.py

次のタプルがあります。

``` python
ages = (35, 42, 67, 34, 46)
```

次の実行結果となるようにPythonプログラムを作成してください。

### 実行結果

``` 
$ python tuple_ex1.py 
67
```

> 前から3番目の要素を出力します。
